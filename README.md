# Proyecto 2 — Neo4j

Inicio del Proyecto 2 basado en Neo4j. Este repositorio contiene la estructura inicial, módulo de conexión a Neo4j, un esqueleto para importación de datos y tareas pendientes para que el equipo continúe.

Resumen rápido:
- Estructura del proyecto en `src/neo4j_project`
- Módulo de conexión: `neo4j_conn.py`
- Esqueleto de importador: `importer.py`
- Documentación y tareas para el equipo en `docs/TODO_FOR_TEAM.md`

Siguientes pasos (priorizados):
1. Revisar las instrucciones completas en `docs/Proyecto 2 - Neo4j.pdf` y el PDF anotado.
2. Rellenar `src/neo4j_project/config.py` con las credenciales de Neo4j (usar variables de entorno).
3. Implementar importador de datos con los mapeos requeridos por la especificación del proyecto.

Ver `docs/TODO_FOR_TEAM.md` para tareas detalladas.
