Cypher Queries

6 consultas parametrizables sobre el grafo de cadena de suministros

Q1 · Products by Category

MATCH (p:Product) WHERE p.categoria = $cat RETURN p

Q2 · Top Suppliers by Rating

MATCH (s:Supplier) RETURN s ORDER BY s.rating DESC LIMIT $n

Q3 · Pending Orders

MATCH (o:OrderCompra) WHERE o.estado = 'PENDIENTE' RETURN o LIMIT $n

Q4 · Transport Status

MATCH (t:Transporte) RETURN t ORDER BY t.fecha_salida LIMIT $n

Q5 · Inventory Status for Product

MATCH (p:Product)-[:ALMACENADO_EN]->(i:Inventory) WHERE p.id_producto = $id RETURN i, p

Q6 · Aggregation Query

MATCH (n:<Label>) RETURN func(n.property)
