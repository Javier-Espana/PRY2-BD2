Cypher Queries

6 consultas parametrizables sobre el modelo de cadena de suministros — ejecuta y ve los resultados

Q1Products by Category
MATCH (p:Product) WHERE p.categoria = $cat RETURN p
Q2Top Suppliers by Rating
MATCH (s:Supplier) RETURN s ORDER BY s.rating DESC LIMIT $n
Q3Pending Orders
MATCH (o:OrderCompra) WHERE o.estado='PENDIENTE' RETURN o LIMIT $n
Q4Transport Status
MATCH (t:Transporte) RETURN t ORDER BY t.fecha_salida LIMIT $n
Q5Inventory Status for Product
MATCH (p:Product)-[:ALMACENADO_EN]->(i:Inventory) WHERE p.id_producto=$id RETURN i
Q6Aggregation Query
MATCH (n:<Label>) RETURN <FUNC>(n.<property>)
